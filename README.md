# arabic-warehouses
1️⃣ إعداد مستودع GitHub للمشروع

قبل أن نبدأ، تأكد من أنك لديك Git مثبت على جهازك وأنك مسجل الدخول في GitHub.

🔹 الخطوات لإنشاء مستودع GitHub جديد:
	1.	اذهب إلى GitHub وسجل الدخول.
	2.	إنشاء مستودع جديد:
	•	اضغط على زر New Repository.
	•	اختر اسمًا مناسبًا للمشروع (مثلاً: arabic-warehouses).
	•	اجعل المستودع عامًا أو خاصًا حسب رغبتك.
	•	حدد خيار Add a README file.
	•	اضغط Create Repository.

🔹 ربط المستودع بـ VS Code:
	1.	افتح VS Code وانتقل إلى المجلد الذي تريد العمل عليه.
	2.	افتح Terminal في VS Code، ثم اكتب الأوامر التالية:

git init
git remote add origin https://github.com/motaz3d/arabic-warehouses.git
git branch -M main
git pull origin main


	3.	الآن أصبح المستودع جاهزًا للرفع.

2️⃣ كتابة README الأولي للمشروع

سأقوم بإنشاء ملف README.md لك، يحتوي على المعلومات الأساسية حول المشروع.
